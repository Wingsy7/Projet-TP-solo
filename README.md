# Projet TP Solo

Small C# practice repository used to consolidate core .NET and object-oriented programming concepts through short console exercises.

## Content

- `tp1.cs`: introductory notes about C#, .NET, CLR, IL and JIT
- `tp2.cs`: variables, operators, conditions, loops, methods and `ref`
- `tp3.cs`: classes, constructors, encapsulation, inheritance, polymorphism, abstract classes and interfaces

## Why This Repository Exists

This repository is intentionally lightweight. It acts as a progressive learning notebook rather than a full application.

It shows:

- regular practice on language fundamentals
- commented examples written for understanding, not just execution
- progression from syntax basics to object-oriented concepts

## Tech

- C#
- .NET console-style exercises

## Note

This repo is best read as a compact learning progression before larger C# or .NET projects.
